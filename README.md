# Project Title

BärnHäckt BärenBatzen

## Description
This is our backend solution for the BEKB-Challenge. The backend is developed for a frontend which is based on React-Native.
The framework we used is ExpressJS.
## Getting Started

### Dependencies

* Node.JS
* NPM
* mongoDb

### Installing

```
cd code
npm install
```

### Executing program
```
npm start
```
